Python 與 VS Code 安裝設定指南

Windows 篇

歡迎來到 Python 的世界！這份文件將引導你一步步在 Windows 電腦上完成 Python 和專業編輯器 Visual Studio Code (VS Code) 的安裝與設定。跟著以下步驟，你很快就能開始你的第一趟 Python 程式設計之旅。

Part 1: 安裝 Python

Python 是一種功能強大且語法簡單的程式語言，是許多人入門程式設計的首選。

步驟 1: 下載 Python 安裝程式

  1. 前往 Python 官方網站：https://www.python.org/downloads/
  2. 網站會自動偵測你的作業系統 (Windows)，你應該會看到一個黃色的 "Download Python x.x.x" 按鈕。直接點擊它來下載最新穩定版的安裝程式。
畫面描述： Python 官網首頁會以醒目的黃色按鈕顯示最新的 Python 下載連結，例如 "Download Python 3.12.4"。

步驟 2: 執行安裝程式

  1. 下載完成後，打開 .exe 安裝檔。
  2. 在安裝畫面中，務必勾選 左下角的 "Add Python to PATH" 選項。這一步非常重要，它能讓你在電腦的任何地方都能執行 Python 指令。
  3. 接著，直接點選 "Install Now" 進行標準安裝即可。
畫面描述： Python 安裝程式視窗的底部有兩個核取方塊。請務必勾選寫有 "Add Python to PATH" 的那一個，然後再點擊 "Install Now"。

步驟 3: 驗證安裝是否成功

  1. 按下鍵盤上的 Win 鍵 (有 Windows 圖示的按鈕)，直接輸入 cmd 然後按下 Enter，打開「命令提示字元」視窗。
  2. 在黑色視窗中，輸入以下指令並按下 Enter： 
    python --version
  3. 如果安裝成功，畫面會顯示你剛剛安裝的 Python 版本號 (例如 Python 3.12.4)。
  4. 接著，輸入以下指令來檢查套件管理工具 pip 是否也安裝成功： 
    pip --version
    同樣地，你會看到 pip 的版本資訊。
畫面描述： 在黑色的「命令提示字元」視窗中輸入 python --version 後，如果安裝成功，下一行會顯示出 Python 的版本號碼。

如果以上兩個指令都能成功顯示版本號，恭喜你，Python 已經安裝完成了！

Part 2: 安裝 Visual Studio Code (VS Code)

VS Code 是一款由微軟開發的免費、強大的程式碼編輯器。它擁有豐富的擴充功能，是目前最受開發者歡迎的工具之一。

步驟 1: 下載 VS Code

  1. 前往 VS Code 官方網站：https://code.visualstudio.com/
  2. 點擊藍色的 "Download for Windows" 按鈕來下載安裝程式。
畫面描述： VS Code 官網頁面中央會有一個顯眼的藍色按鈕，上面寫著 "Download for Windows"。

步驟 2: 執行安裝程式

  1. 執行下載好的安裝程式。
  2. 在安裝過程中，你可以依照預設選項點擊「下一步」。特別建議在「選取附加工作」的頁面，勾選以下兩個選項，這會讓你未來使用上更方便：
    • "將'以 Code 開啟'操作新增至 Windows 檔案總管檔案右鍵功能表"
    • "將'以 Code 開啟'操作新增至 Windows 檔案總管目錄右鍵功能表"
  3. 完成後，啟動 VS Code。

Part 3: 設定 VS Code for Python 開發

安裝好 VS Code 後，我們需要做一些簡單設定，讓它成為強大的 Python 開發工具。

步驟 1: 安裝 Python 擴充功能

  1. 在 VS Code 左側的活動列中，點擊看起來像方塊堆疊的圖示，進入「擴充功能」市場。
  2. 在搜尋框中輸入 Python。
  3. 找到由 Microsoft 發行的那個，點擊旁邊的 Install 按鈕進行安裝。這個擴充功能提供了程式碼高亮、自動補全 (IntelliSense)、程式碼除錯等核心功能。
畫面描述： 在 VS Code 側邊欄的擴充功能市集中，搜尋 "Python" 後，選擇由 Microsoft 發行、下載量最高的那一個進行安裝。

步驟 2: 建立並執行你的第一個 Python 程式

  1. 在電腦上建立一個新的資料夾，例如 MyPythonProjects，用來存放你未來的 Python 程式碼。
  2. 在 VS Code 中，點擊左上角的 `檔案 (File)` > `開啟資料夾 (Open Folder...)`，然後選擇你剛剛建立的資料夾。
  3. 在左側的「檔案總管」面板中，點擊新增檔案的圖示，並將檔案命名為 hello.py。.py 是 Python 程式碼的副檔名。
  4. 在 hello.py 的編輯區中，輸入你的第一行 Python 程式碼： 
    print("Hello, Python! 我學會寫程式了！")
  5. 點擊 VS Code 右上角的 綠色播放按鈕 (▶) 來執行你的程式。
  6. 你會看到 VS Code下方彈出一個「終端機 (TERMINAL)」視窗，並印出 "Hello, Python! 我學會寫程式了！" 的結果。
畫面描述： 程式碼編輯完成後，點擊右上角的播放 (▶) 按鈕。VS Code 下半部會出現一個終端機面板，其中會顯示程式的輸出文字 "Hello, Python! 我學會寫程式了！"。

步驟 3: 選擇 Python 解譯器

通常情況下，Python 擴充功能會自動偵測你安裝的 Python 版本。如果 VS Code 沒有自動選取，你可以手動指定：

  1. 按下 Ctrl + Shift + P 打開命令面板。
  2. 輸入 Python: Select Interpreter。
  3. 從列表中選擇你之前安裝的 Python 版本 (通常會標示為 "Recommended")。

Part 4: 注意事項與建議

  • 1.
    檔案儲存與副檔名: 所有 Python 程式碼檔案都必須以 .py 作為結尾，例如 my_script.py。
  • 2.
    養成良好習慣: 每個專案都建立一個獨立的資料夾來管理，不要把所有 .py 檔案都散落在桌面。
  • 3.
    注意 PATH 環境變數: 如果你在 cmd 中輸入 python 卻顯示「不是內部或外部命令」，99% 的原因是在安裝 Python 時忘記勾選 "Add Python to PATH"。此時最簡單的解決方法是移除 Python 後重新安裝一次，並記得勾選該選項。
  • 4.
    善用終端機: VS Code 內建的終端機是你未來的好夥伴，無論是執行程式、安裝套件 (使用 pip install <套件名稱>)，都會在這裡進行。
  • 5.
    保持好奇心: 嘗試修改 print() 裡面的文字，看看會發生什麼事。程式設計就是從不斷的嘗試與修正中學習的。

你已經完成了所有必要的設定，現在可以正式開始你的 Python 學習之旅了。

祝你學習順利，享受編寫程式的樂趣！
