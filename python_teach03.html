前言

在 Python 以及任何程式語言中，「運算子」是執行各種運算的核心符號。它們是語言的動詞，讓我們能夠操作變數和值。無論是進行簡單的數學計算、比較數據的大小，還是進行複雜的邏輯判斷，運算子都扮演著不可或缺的角色。

掌握運算子的使用是您程式設計旅程的第一個重要里程碑。本指南將深入探討 Python 中最核心的七大類運算子，並透過詳細的程式碼解說，幫助您建立穩固的基礎。

1. 算術運算子 (Arithmetic)

算術運算子用於執行你我所熟知的基本數學運算，例如加、減、乘、除等。

範例程式碼與解說

a = 10
b = 3

# 加法 (+)
result_add = a + b
print(f"a + b = {result_add}")

# 除法 (/)
result_div = a / b
print(f"a / b = {result_div}")

# 整數除法 (//)
result_floor_div = a // b
print(f"a // b = {result_floor_div}")

# 餘數 (%)
result_mod = a % b
print(f"a % b = {result_mod}")

# 次方 (**)
result_pow = a ** b
print(f"a ** b = {result_pow}")

第 1-2 行: 我們宣告了兩個整數變數 `a` 和 `b`，它們將作為我們運算的基礎。

第 5 行: 使用加法運算子 `+`，將 `a` 和 `b` 的值相加，結果為 `13`。

第 9 行: 使用除法運算子 `/`，`10` 除以 `3` 的結果是一個浮點數 `3.333...`。

第 13 行: 使用整數除法 `//`，它只會保留商的整數部分，所以結果是 `3`。

第 17 行: 使用餘數運算子 `%`，`10` 除以 `3` 餘 `1`。

第 21 行: 使用次方運算子 `**`，計算 `10` 的 `3` 次方，即 `10 * 10 * 10`，結果為 `1000`。

2. 比較運算子 (Comparison)

比較運算子用於比較兩個值，其運算結果永遠是一個布林值 (`True` 或 `False`)。這在流程控制（例如 `if` 語句）中至關重要。

範例程式碼與解說

x = 5
y = 10

# 等於 (==)
print(f"x 是否等於 y? {x == y}")

# 不等於 (!=)
print(f"x 是否不等於 y? {x != y}")

# 大於 (>)
print(f"x 是否大於 y? {x > y}")

# 小於等於 (<=)
print(f"y 是否小於等於 10? {y <= 10}")

第 1-2 行: 宣告兩個變數 `x` 和 `y` 用於比較。

第 5 行: 使用 `==` 檢查 `x` 和 `y` 的值是否相等。因為 `5` 不等於 `10`，所以結果為 `False`。

第 8 行: 使用 `!=` 檢查 `x` 和 `y` 是否不相等。因為 `5` 的確不等於 `10`，所以結果為 `True`。

第 11 行: 使用 `>` 檢查 `x` 是否大於 `y`。`5` 並沒有大於 `10`，結果為 `False`。

第 14 行: 使用 `<=` 檢查 `y` 是否小於或等於 `10`。`10` 等於 `10`，條件成立，所以結果為 `True`。

3. 邏輯運算子 (Logical)

邏輯運算子 `and`, `or`, `not` 用於組合多個布林表達式，常用於複雜的條件判斷。

範例程式碼與解說

has_ticket = True
is_vip = False

# and: 必須同時滿足兩個條件
can_enter = has_ticket and is_vip
print(f"是 VIP 且有票嗎? {can_enter}")

# or: 滿足任一條件即可
can_get_discount = has_ticket or is_vip
print(f"有票或是 VIP 嗎? {can_get_discount}")

# not: 反轉條件
is_not_vip = not is_vip
print(f"不是 VIP 嗎? {is_not_vip}")

第 1-2 行: 設定兩個布林變數，`has_ticket` 為 `True` (有票)，`is_vip` 為 `False` (不是VIP)。

第 5 行: 使用 `and` 運算子。必須兩個條件都為 `True`，結果才會是 `True`。因為 `is_vip` 是 `False`，所以 `can_enter` 結果為 `False`。

第 9 行: 使用 `or` 運算子。只要其中一個條件為 `True`，結果就是 `True`。因為 `has_ticket` 是 `True`，所以 `can_get_discount` 結果為 `True`。

第 13 行: 使用 `not` 運算子。它會將布林值反轉。`not is_vip` (即 `not False`) 的結果為 `True`。

4. 位元運算子 (Bitwise)

位元運算子直接對整數的二進位表示法進行操作。這在低階系統程式設計、演算法優化或處理硬體封包時非常有用。

範例程式碼與解說

m = 60  # 二進位: 0011 1100
n = 13  # 二進位: 0000 1101

# & (AND): 對應位元皆為 1，結果才為 1
result_and = m & n
print(f"m & n = {result_and}") # 0000 1100 -> 12

# | (OR): 對應位元只要有 1，結果就為 1
result_or = m | n
print(f"m | n = {result_or}") # 0011 1101 -> 61

# << (左移): 將二進位碼向左移動，右邊補 0
result_left_shift = m << 2
print(f"m << 2 = {result_left_shift}") # 1111 0000 -> 240

第 1-2 行: 宣告 `m` 和 `n`。`m` 的二進位是 `00111100`，`n` 是 `00001101`。

第 5 行: `&` (AND) 運算。比較 `m` 和 `n` 的每個位元，只有當兩個位元都是 `1` 時，結果的對應位元才是 `1`。結果是 `00001100`，即十進位的 `12`。

第 9 行: `|` (OR) 運算。只要兩個位元中有一個是 `1`，結果的對應位元就是 `1`。結果是 `00111101`，即十進位的 `61`。

第 13 行: `<<` (左移) 運算。將 `m` 的二進位表示向左移動 `2` 位，右側空出的位置補 `0`。這相當於將原數值乘以 `2` 的 `2` 次方。結果是 `11110000`，即十進位的 `240`。

5. 指派運算子 (Assignment)

指派運算子用於將值賦予變數。除了基本的 `=`，還有結合算術運算子的複合指派運算子，可讓程式碼更簡潔。

範例程式碼與解說

counter = 100
print(f"起始 counter: {counter}")

# 加法指派 (+=)
counter += 10  # 等同於 counter = counter + 10
print(f"執行 += 10 後: {counter}")

# 乘法指派 (*=)
counter *= 2   # 等同於 counter = counter * 2
print(f"執行 *= 2 後: {counter}")

第 1 行: 初始化變數 `counter` 的值為 `100`。

第 5 行: `counter += 10` 是一個複合指派。它會先執行 `counter + 10` (即 `100 + 10`)，然後將結果 `110` 重新指派給 `counter`。

第 9 行: `counter *= 2` 同樣是複合指派。此時 `counter` 的值是 `110`，執行 `110 * 2` 後，將結果 `220` 重新指派給 `counter`。

6. 身分運算子 (Identity)

身分運算子 `is` 和 `is not` 用於比較兩個變數是否指向記憶體中**同一個物件**，而不僅僅是它們的值相等。

`==` 檢查的是「值」是否相等 (value equality)。
`is` 檢查的是「身分」是否相同 (identity equality)，即記憶體位址是否相同。

範例程式碼與解說

list_a = [1, 2, 3]
list_b = [1, 2, 3]
list_c = list_a

print(f"list_a == list_b: {list_a == list_b}")
print(f"list_a is list_b: {list_a is list_b}")
print(f"list_a is list_c: {list_a is list_c}")

第 1-3 行: `list_a` 和 `list_b` 是兩個獨立的列表物件，它們的內容恰好相同。`list_c` 則是 `list_a` 的一個引用，它們指向同一個物件。

第 5 行: `list_a == list_b` 比較的是「值」。因為兩個列表的內容完全相同，所以結果為 `True`。

第 6 行: `list_a is list_b` 比較的是「身分」。儘管內容相同，但它們是儲存在記憶體中不同位置的兩個物件，所以結果為 `False`。

第 7 行: `list_a is list_c` 的結果為 `True`，因為 `list_c` 只是 `list_a` 的另一個名字，它們指向的是記憶體中完全相同的物件。

7. 成員運算子 (Membership)

成員運算子 `in` 和 `not in` 用於測試一個值是否存在於一個序列（如 `list`, `tuple`, `string`）中。

範例程式碼與解說

courses = ["Math", "Physics", "Computer Science"]

print(f"'Physics' in courses: {'Physics' in courses}")
print(f"'History' not in courses: {'History' not in courses}")

第 1 行: 建立一個包含三個課程名稱的列表 `courses`。

第 3 行: 使用 `in` 運算子檢查字串 `"Physics"` 是否是 `courses` 列表中的一個成員。因為它存在，所以結果為 `True`。

第 4 行: 使用 `not in` 運算子檢查字串 `"History"` 是否「不」在 `courses` 列表中。因為它的確不存在，所以結果為 `True`。
